Responsive Email Template

A responsive email template, based on Phil Wareham's Responsive Email template (http://github.com/philwareham/responsive-email) with SalsaLabs merge tags in place and a wide banner image.  

The altered design for a wide banner image has not gone through extensive testing, but responds well in iphone 4, iphone 5 and Android 4.0 and 4.1 (HTC).

This version was created by CWA.
